RoboFriends React

This React app was developed as part of my Complete Web Developer course. It's a stateful web application that leverages a few external APIs to provide user information and images. The user data comes from JSONPlaceholder, while the images are sourced from RoboHash. To create the page layout, Tachyons framework was used. This project marks my first experience with React.js, which made it simple yet responsive. I started with the create-react-app package and wrote the code in JavaScript/JSX, following the course. The search bar and title remain visible no matter the screen size, and the number of cards across adjusts dynamically to the screen width. Additionally, the app only displays cards that match the search query. If you're interested in checking out the code, head over to my GitHub repository.
